# KinmuCore ドキュメント

KinmuCore勤怠管理システムの各種マニュアルです。

---

## 📚 マニュアル一覧

### 🚀 [クイックスタートガイド](./QUICK_START.md)
**所要時間:** 15分  
**対象:** 初めて使用する管理者

初期設定から打刻テストまで、最短で使い始めるための手順書です。

---

### 👨‍💼 [管理者向け操作マニュアル](./ADMIN_MANUAL.md)
**対象:** 管理者・店長

日常的な運用に必要な以下の操作方法を説明しています：

- ログイン方法
- 店舗管理
- スタッフ管理
- ポリシー管理
- 勤怠管理
- 打刻画面の設定
- データ出力
- トラブルシューティング

---

### 👥 [スタッフ向け操作マニュアル](./STAFF_MANUAL.md)
**対象:** 打刻を行うスタッフ全員

タブレットでの打刻方法を簡潔に説明しています：

- 出勤打刻
- 休憩開始/終了
- 退勤打刻
- よくある質問

💡 **印刷推奨:** タブレット近くに掲示すると便利です

---

### ⚙️ [セットアップマニュアル](./SETUP_MANUAL.md)
**対象:** システム管理者・IT担当者

システムの初期構築と保守管理の手順書です：

- システム要件
- 初期セットアップ
- 管理者ユーザー作成
- タブレット設定
- セキュリティ設定
- バックアップと復旧
- メンテナンス手順
