# KinmuCore クイックスタートガイド

**所要時間:** 約15分  
**対象:** 初めてKinmuCoreを使用する管理者

---

## ✅ このガイドで実現できること

15分後、あなたは以下ができるようになります：

1. ログインして管理画面にアクセス
2. 店舗とスタッフを登録
3. タブレットで打刻できる状態にする

---

## ステップ1: ログイン（1分）

1. ブラウザで https://kinmucore.vercel.app を開く
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

✅ **確認:** ダッシュボード画面が表示されればOK

---

## ステップ2: 店舗を登録（2分）

1. サイドバーの「**店舗管理**」をクリック
2. 「**+ 新規店舗**」ボタンをクリック
3. 以下を入力：
   ```
   店舗名: 本店
   住所: （任意）
   電話番号: （任意）
   ```
4. 「**登録**」をクリック

✅ **確認:** 店舗一覧に「本店」が表示されればOK

---

## ステップ3: ポリシーを作成（3分）

1. サイドバーの「**ポリシー管理**」をクリック
2. 「**+ 新規ポリシー**」ボタンをクリック
3. 以下を入力：
   ```
   ポリシー名: 標準ルール
   対象店舗: 本店
   締め日: 月末
   丸め単位: 15分単位
   休憩控除: 手動（打刻ベース）
   ```
4. 「**登録**」をクリック

✅ **確認:** ポリシー一覧に「標準ルール」が表示されればOK

---

## ステップ4: スタッフを登録（5分）

1. サイドバーの「**スタッフ管理**」をクリック
2. 「**+ 新規スタッフ**」ボタンをクリック
3. 以下を入力：
   ```
   氏名: 山田 太郎
   所属店舗: 本店
   雇用形態: 正社員
   ```
4. 「**登録**」をクリック

**繰り返し:** 他のスタッフも同様に登録

💡 **ヒント:** 後から追加・編集できるので、まずは1〜2名登録すればOK

✅ **確認:** スタッフ一覧に登録した名前が表示されればOK

---

## ステップ5: タブレットを設定（4分）

### 打刻URLを取得

1. サイドバーの「**店舗管理**」をクリック
2. 本店の「**打刻ページ**」リンクをクリック
3. 表示されたURLをコピー

### タブレットで開く

1. タブレットのブラウザでコピーしたURLを開く
2. スタッフ一覧が表示されることを確認

### ホーム画面に追加（iPad/iPhone）

1. Safari下部の「共有」ボタンをタップ
2. 「ホーム画面に追加」をタップ
3. 名前を「勤怠打刻」に変更して追加

✅ **確認:** タブレットのホーム画面にアイコンが表示されればOK

---

## ステップ6: 打刻テスト（1分）

### 実際に打刻してみる

1. タブレットで「勤怠打刻」アイコンをタップ
2. 登録したスタッフ名（例：山田 太郎）をタップ
3. 「**出勤**」ボタンをタップ

✅ **確認:** 「出勤を記録しました」と表示されればOK

### 管理画面で確認

1. PCの管理画面に戻る
2. サイドバーの「**勤怠管理**」をクリック
3. 今日の日付に打刻記録が表示されていることを確認

✅ **確認:** 山田 太郎の出勤時刻が記録されていればOK

---

## 🎉 完了！

お疲れ様でした！基本的なセットアップが完了しました。

## よくある質問

### Q: スタッフ数が多い場合は？

**A:** Excel等で一覧を作成してから、1つずつ登録するとスムーズです。
将来のバージョンでCSV一括インポート機能を追加予定です。

### Q: 複数店舗がある場合は？

**A:** 店舗ごとに「店舗登録」→「ポリシー作成」→「スタッフ登録」を繰り返してください。
各店舗ごとに異なるルール（ポリシー）を設定できます。

### Q: 打刻画面のURLを忘れた場合は？

**A:** 管理画面の「店舗管理」からいつでも再確認できます。
